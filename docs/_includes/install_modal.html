Install Eventific

Command Side

Using npm:

$ npm i @eventific/command @eventific/core

Or yarn:

$ yarn add @eventific/command @eventific/core

Query Side

Using npm:

$ npm i @eventific/query @eventific/core

Or yarn:

$ yarn add @eventific/query @eventific/core

Saga

Using npm:

$ npm i @eventific/saga @eventific/core

Or yarn:

$ yarn add @eventific/saga @eventific/core
