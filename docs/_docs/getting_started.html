Getting Started

A guide to help you setup Eventific in no time

Installation

This package can either be installed through Npm or Yarn. Here is a list of commands for each service type. The three main service types can only be run on node. The core package however that contains the fundamental construct can be used in the browser

Command Side

The command side takes care of incoming commands and turns them into events. Read more about command sides

Using Npm

$ npm i --save @eventific/command @eventific/core

Using Yarn

$ yarn add @eventific/command @eventific/core

Query Side

Query Side is how you fetch data from your system it handles prebuilt views of the current state than might be joined between aggregates. Read more about query sides

Using Npm

$ npm i --save @eventific/query @eventific/core

Using Yarn

$ yarn add @eventific/query @eventific/core

Saga

Saga is how you handle side effects and aggregate coordination. It can be triggered by different types of external sources such as time or events. Read more about sagas

Using Npm

$ npm i --save @eventific/saga @eventific/core

Using Yarn

$ yarn add @eventific/saga @eventific/core
